## 用自己的话回答

- 什么是特征，什么是标签：特征（Feature）：用来描述样本属性、用来做判断依据的输入数据。比如预测房价时，房屋面积、楼层、房龄都是特征；特征是模型的输入。
标签（Label）：模型最终要预测的目标结果，属于输出值。对应上面例子里的房屋成交价格；分类任务里猫狗图片中“猫/狗”类别就是标签。

- 为什么要保留测试集：（1）.训练集专门用来给模型拟合、学习数据规律；验证集用来调参；测试集全程不参与训练与调优。

（2）. 用来模拟真实线上未知数据，客观检验模型泛化能力，判断模型是否在陌生数据上依旧有效。

（3）. 提前发现过拟合问题，避免模型在训练数据表现极好、真实场景效果很差。

- 为什么83%准确率仍可能没有用：1. 样本类别极度不均衡：例如90%样本都是正常数据，10%为故障数据。模型无脑预测全部为正常，就能轻松拿到90%左右准确率，看似83%准确率很高，但完全识别不出故障样本，业务完全失效。

2. 任务核心指标不是准确率：医疗疾病诊断、风控诈骗识别场景，漏诊/误判代价极高，宁可降低总体准确率，也要提升召回率，单纯准确率无业务意义。

3. 过拟合导致泛化差：83%准确率来自训练集，测试集准确率暴跌，模型无法适配新数据，上线无法使用。

4. 数据分布偏移：测试集和未来真实业务数据分布不一致，准确率不具备参考价值。
